🔒  Zero-Knowledge Token Design

Login codes (OTPs) and session tokens are stored only as cryptographic hashes — never in plaintext. Even a full database exposure would yield nothing usable.

📋  Versioned Consent Records

Every user's acceptance of the Privacy Statement and Terms of Use is recorded with an exact version identifier and timestamp — creating a permanent, auditable consent trail.

Security
Defence in Depth
Security is enforced at every layer independently. A failure at one layer does not cascade through the system.
🏠

Cloud-Native Infrastructure

Hosted entirely on Google Cloud Platform. Stateless compute, managed database, IAM-controlled storage. No on-premise servers, no exposed admin consoles.

🔑

No Passwords, Ever

Authentication uses one-time codes delivered to a verified phone or email — eliminating the largest class of credential attacks entirely. Short-lived tokens expire within the hour.

🏠

Strict Data Isolation

Athletes see only their own data. Clinical staff see only their organisation's athletes. The AI inference engine is never reachable by any external client — only the backend can invoke it.

📷  Thermal images never travel through the API

The NUR Sensor uploads infrared scans directly to Google Cloud Storage via a time-limited signed URL. The image binary never passes through the application server — eliminating an entire category of interception risk. The storage bucket has no public access; only IAM-bound service accounts may read it.
